Device Maintenance ​

Although the HIG Glove was designed with maximum lifespan in mind, please follow these instructions for storage and cleaning to extend its service life as much as possible:

Storage Requirements ​

  • Light conditions: Avoid direct sunlight
  • Temperature: 5°C to 30°C
  • Humidity: 30% to 60%
  • Battery level: Before storage, ensure the Power Button LED shows Orange

Tip

  • Using the original HIG Glove packaging box for storage effectively blocks sunlight and dust (ensure the lid is closed)
  • Avoid prolonged storage in these states:
    • High battery: Power Button LED Green
    • Critically low battery: Power Button LED Flashing Red
  • After long-term storage (e.g., over 6 months), perform a full charge-discharge cycle before initial use.

Cleaning the Glove ​

Danger

  • Before washing, ensure all electronic parts are removed from the HIG Glove! Otherwise, irreversible damage will occur!

Removing Parts ​

--
1. Use stickers to mark each Hall Effect Module to avoid subsequent installation errors 2. Remove all Magnet Modules and Controller Module (If you have)
3. Remove Data Box using PH 00 or H 1.3 screwdriver (version-dependent) 4. Finish

Washing Requirements ​

  • Washing method: Hand wash only
  • Prohibited actions:
    • Machine washing
    • Bleaching
    • Tumble drying
  • Drying method: Lay flat to air-dry in a cool, well-ventilated area
  • Temperature limit: Water temperature ≤ 40°C
